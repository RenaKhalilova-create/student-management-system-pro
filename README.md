# Student Management System (C++)

Console-based application for managing student records with file persistence.

## Features
- Add new students
- View all students
- Search students by name
- Delete students
- Save data to file
- Load data from file

## Technologies
- C++
- File handling (fstream)
- STL (vector)

## How to Run
1. Compile the code:
   g++ main.cpp -o app

2. Run the program:
   ./app

## Project Structure
- main.cpp — main logic of the program
- students.txt — file where data is stored

## Description
This project demonstrates core programming concepts such as data structures, file handling, and basic CRUD operations in C++.
